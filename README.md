[Message](https://discord.com/channels/850574546439766047/1501679552525504663) from resource pack author ic22487:

If you are making your own language translation overlay, you are free to upload it to Modrinth as long as the following conditions are met:
 - Your overlay resource pack has Hypixel+ marked as a dependency
 - Your overlay resource pack does not contain any Hypixel+ assets, only the updated lang file(s)
 - Your overlay resource pack is uploaded with a [CC BY-NC-SA 4.0 license](https://creativecommons.org/licenses/by-nc-sa/4.0/)
 - You have replaced `English (US) Translation by ic22487` in `hplus.about.translator.hover` with your own name and language you have translated the pack into.
 - The version of the lang file you are using is not from an ongoing donator only beta

And some advice:
`toggle.on.negative`, `toggle.on.balance`, `toggle.off.negative`, and `toggle.off.balance` can be used to balance out the icon spacing used in the Ko-Fi tab (You will find this important if you need to change the on/off toggle text)

If you want your overlay added to the base pack, DM me the lang file and I'll consider adding it!
Thank you!
